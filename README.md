# Poker Game App (**Status: Work-in-Progress**)
A poker game for research development.

## Technology Stack
+ Babel v7
+ ESLint v6
+ Jest v25
+ Material UI v4
+ Parcel v1
+ Redux v4
+ React v16

## Install (Dev)
### Requires
+ Node.js
+ yarn or npm

### Run a dev server
```sh
yarn start
```

Or

```sh
npm run start
```
